# рџ“Ѓ Contribution Guidelines

Please follow these guidelines when contributing.

## 1. Organize Your Files

Place your resources inside the appropriate technology folder.

Example:

```
React/
    React Interview Questions.pdf

Java/
    OOP Notes.pdf

Go/
    Go Cheat Sheet.md
```

If a folder doesn't exist, feel free to create one.

---

## 2. Use Meaningful File Names

Good examples:

```
React_Interview_Questions.pdf
Java_Collections_Notes.md
Docker_Cheat_Sheet.pdf
```

Avoid names like:

```
Notes.pdf
File1.pdf
Document.pdf
```

---

## 3. Keep Resources Educational

Please contribute resources that help others learn.

Avoid:

- Spam
- Pirated content
- Unrelated files
- Duplicate resources

---

## 4. Respect Copyright

Only upload:

- Your own work
- Public domain resources
- Openly licensed material
- Resources you have permission to share

Always credit the original author whenever required.
